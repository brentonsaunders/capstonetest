What is CR Comment Analyzer?

The CR Comment Analyzer is a tool for CRUX users to assess the quality of their code review feedback. It allows users to submit code review comments for analysis by Amazon Comprehend, and then to retrieve the sentiment of the comment text along with its key phrases. This tool is useful for developers to track whether they are providing polite and actionable feedback to their teams.
